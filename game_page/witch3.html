系列新作《巫師 3：狂獵》是一款擁有龐大開放世界的角色扮演遊戲，玩家將化身自幼接受訓練、歷經變種，並累積超乎常人的戰鬥技巧、力量和反應的狩魔獵人，在探索道德淪喪的奇幻世界中冒險，找尋預言之子的蹤影。

配備需求
     OS: Windows 7 or 8 (8.1) 64 位元以上
     DirectX : 11或更新的版本
     HDD : 40 GB以上
     相容Windows鍵盤/滑鼠/搖桿
最低:
     CPU:Intel Core i5 2500K 3.3GHz / AMD Phenom II X4 940
     GPU:Nvidia GeForce GTX 660 / AMD Radeon HD 7870
     VRAM:2 GB
     RAM:6GB
建議:
     CPU:Intel  Core i7 3770 3.4 GHz / AMD AMD FX-8350 4 GHz
     GPU:Nvidia GeForce GTX 770 / AMD Radeon R9 290
     VRAM:4 GB
     RAM:8GB
